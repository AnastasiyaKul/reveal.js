What is it?

Typescript is a superset of JavaScript.

It means any JS code is correct from the point of view of TypeScript. But TypeScript has some additional features that are not included in JavaScript.

Why does it interesting to us?

  • Advanced features
  • It used as base for popular frameworks
  • Stable development of the language
